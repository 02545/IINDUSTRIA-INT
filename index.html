Descripción del Proyecto

Este proyecto consistió en una maqueta interactiva sobre un Tinaco colocando sensores para saber el llenado y vacío de este mismo, al igual saber el nivel de agua, se integro el circuito con Labview y ardiuno

Galería de Imágenes

La casa se realizo en Solidworks.

Materiales Utilizados

  • Capacitor 1000 micriF
  • Modulo Relevador
  • Resistencia 560ohms
  • Transistor 2n2222a
  • Sensor lm35
